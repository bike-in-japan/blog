1. How to create a post

To create a new blog post via GitHub Issues:

Open an Issue: Create a new issue in this repository.
    Title: The issue title will be used as the post title.
    Content:
        Tags: Add a line like 
        Categories: Add a line like 
        GPX Track: To include an interactive map:
            Upload your .gpx file to assets/gpx/.
            Add a line in the issue body: gpx: assets/gpx/your-track.gpx.
        Body: The rest of the issue body will be the post content.
